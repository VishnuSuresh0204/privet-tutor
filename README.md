# Online Private Tutor

A premium web application for managing private tutoring sessions, bookings, and student-teacher interactions.

## Features

- **Tutor Dashboard**: Manage active tutoring sessions with a modern, card-based interface.
- **Booking Management**: Approve or reject student booking requests seamlessly.
- **Status Tracking**: Visual indicators for session status (Pending, Approved, Demo Requested, Video Uploaded).
- **Communication**: Integrated chat and study notes sharing.
- **Responsive Design**: Premium, mobile-friendly UI with high-contrast elements for better visibility.

## Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML5, Vanilla CSS3, Bootstrap 4
- **Database**: SQLite (Development)
- **Icons**: Font Awesome 5

## UI Enhancements (Recent)

- **Standardized Navbar**: Improved visibility with a semi-transparent dark background.
- **Modern Cards**: Sophisticated card-based layout for bookings with student avatars and improved hierarchy.
- **Hero Sections**: Premium hero banners with dark overlays for better text legibility.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/VishnuSuresh0204/privet-tutor.git
   ```
2. Set up a virtual environment and install dependencies (if requirements.txt is provided).
3. Run migrations:
   ```bash
   python manage.py migrate
   ```
4. Start the development server:
   ```bash
   python manage.py runserver
   ```
